# Banking Dashboard

## Project Overview

This project is an interactive Banking Dashboard developed using Power BI and MySQL. It demonstrates the complete data analysis process, including data cleaning, transformation, exploratory data analysis (EDA), and dashboard creation.

---

## Workflow

Dataset → MySQL → Data Cleaning → EDA → Power BI Dashboard

---

## Tools Used

* Power BI
* MySQL
* SQL
* DAX
* Power Query

---

## Features

* Data Cleaning and Transformation
* Exploratory Data Analysis (EDA)
* Loan Analysis
* Deposit Analysis
* Customer Analysis
* Interactive Dashboard

---

## Dashboard Pages

### Home

Provides an overview of the banking dataset with key metrics and summary visuals.

### Loan Analysis

Shows loan distribution, customer segments, and loan-related insights.

### Deposit Analysis

Analyzes deposits, account balances, and branch performance.

### Summary

Presents the main business insights and overall findings.

---

## Key Insights

* High-income customers tend to have higher deposits.
* Savings and checking account balances show a positive relationship.
* Branch performance varies across different locations.
* Customer demographics help identify banking trends.
